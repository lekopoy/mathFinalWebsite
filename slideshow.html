Computer Bases

Computer Bases are a fundemental part of our society, with their being—

Decimal

Decimal(Base-10) is the traditional number system used by humans, and it is very important to arithmetic. It features numbers 0-9, allowing you to do math such as 1+1=2. This isn't used directly by computers but is used for humans to understand computers.

Hexadecimal

Hexadecimal(Base-16), is a more enriched yet confusing positional numeral system. It features numbers 0-9 same as Base-10 and includes the letters A to F in the alphabet, representing digits 10-15, like this 0x4A = 74. Hexadecimal is massively more present in computers than Base-10.

Binary

Binary(Base-2) is the most basic numeral system, it uses the digits 0 & 1. A number can be represented in 8-bit binary(BYTES) like this 00111111 = 63. It's also rarely used in anything but computer science.

Binary

Invention of the Modern Computer

Konrad Zuse is regarded as the inventor of the modern computer. Now the term "modern computer" does not refer to our current computers, however it refers to the introduction of binary and boolean logic into computers. Computers had previously been incorporating Base-10 into the machine; leading to much more complicated systems. This event is exetremely historical significant as it directly led to the modern day computers. Nowadays we use computers for everything, whether it be—banking, government systems, medicine development, entertainment, warfare, space exploration and much more.

Boolean Algebra

Boolean Algebra is the mathematics applied to Base-2 allowing for arithmetic in Base-2. It was introduced by George Boole back in 1847, and has since been expanded upon. We use Boolean Algebra with Transistors to create Logic Gates. Logic gates have multiple types with—AND, OR & NAND being some of the more prominent ones. Inside a modern day computer lies hundreds of millions to billions of Logic Gates. If an electricial current is flowing then it is On/True which is 1 in binary, if it's not flowing then it's Off/False which is 0. These individual states of electricial flow are called bits.

Converting to Binary

byte = 8 bits. E.g. 01010101 2^7 2^6 2^5 2^4 2^3 2^2 2^1 2^0 are the values for each bit in a byte. So in our example 01010101 we can raise each 1 to a certain 2 exponent. 64 + 16 + 4 + 1 = 85. Also, most modern systems run on 64-bit integer systems, so instead of 01010101 we get 0000000000000000000000000000000000000000000000000000000001010101.

Hexadecimal

Use in Computer Science

Base-16 has been around for a very long time, and it's been used in computers almost as long as the implementation of binary in computers. We use hexadecimal to make code more readable for humans inside low-level languages for a better understanding of what you're doing when directly controlling hardware. Hexadecimal is used in assembly languages like ARM / AArch64 Assembly and x86-64 / x64 Assembly.

Hexadecimal Numbers

When we take a number in Base-10 like 52, what we are actually doing is 5 * 10^1, and 2 * 10^0. In Base-2 with a number like 1011, we'd do 1 * 2^3, 0 * 2^2, 1 * 2^1, 1 * 2^0, and we'd get 11. In Base-16 not only do we have the extra symbols A-F(10-15), but we raise every digit to a power of 16. So 7AC2 would be 7 * 16^3, 10 * 16^2, 12 * 16^1, 2 * 16^0, which is 31426.

Converting to Hexadecimal

For this example we will take the number 151 and convert it to hexadecimal. To start we will divide 151 by 16, we get 9 with a remainder of 7. Next we divide the new quotient(9) by 16, 9/16 = 0 remainder 9. The first remainder is the rightmost digit, and the second remainder is the leftmost digit. You then get 97, which we can write as 0x97.

Decimal

Origin

All of Base-10 can be traced back to ancient civilizations. The Egyptian numerals, Brahmi numerals, Greek numerals, Hebrew numerals, Roman numerals, and Chinese numerals all use decimal(Base-10). A common belief is it originates from the ten fingers on our hands, and people counted with their fingers.

Basic Mathematics

Base-10 is the primary system used in standard mathetmatics. We use it in this class, to perform math we can use the four basic operators, * / + -. It allows us to perform equations like 4 * 3 = 12, 12/2 = 6, 1 + 1 = 2, and 8-5 = 3. We can expand this with BEDMAS, (2+2) * 4² in which we first solve the brackets (4) * 4², then the exponent 4 * 16, and finally it's just multiplying 4 * 16 = 64.

Usage in Computer Science

Actual Base-10 is commonly used in high-level coding languages to help make code more readable for humans. E.g. Arrays: myArray = ["hello", "hi", "e"]. Now inside arrays you can access specific values by using the index position like this myArray[1]. Now if I were to print this to the output print(myArray[1]) you might assume it will print "hello", however it will actually output:

hi

This is because the index position starts at 0, not 1; as Base-10 is the numbers 0-9.
